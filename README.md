# 패션 정보 공유 웹사이트 (대학 전공 프로젝트)

본 프로젝트는 대학교 웹 프로그래밍 수업의 기말 과제로 제작된 1인 프로젝트입니다.

# 확인 시 주의사항
현재 리포지토리에 업로드된 사진 파일은 로컬 경로로 설정되어 있어, 최신 커밋 버전에서는 이미지가 정상적으로 출력되지 않을 수 있습니다. 

**소스 코드의 구조(HTML/CSS)를 확인하시려면 아래 절차를 권장합니다.**
1. 상단의 [Commits] 탭 클릭
2. **'first commit'** 버전의 파일을 확인

# 주요 기능
- 직관적인 패션 쇼핑몰 UI/UX 구현
- HTML5 시멘틱 태그를 활용한 웹 표준 준수
- 효율적인 레이아웃 구성을 위한 CSS 활용
